# Release Notes

## v0.1.1 - 2026-02-20

### 기능

- `:` 단축키로 명령 실행 팝업을 열고, 활성 패널 경로 기준으로 쉘 명령 실행 지원

### 버그 수정

- 라이브러리 업데이트에 따른 7z 처리 호환 문제 수정
- 메인 루프 타입 호환 이슈 수정

## v0.1.0 - 2026-02-18

첫 공개 버전입니다.

### 주요 내용

- 터미널 듀얼 패널 파일 매니저 기본 워크플로 제공
- 키보드 중심 탐색 및 파일 작업 지원
- 파일 선택/다중 선택/선택 반전 지원
- 필터, 정렬, 숨김 파일 토글 기능 제공
- 탭, 히스토리, 북마크 기능 제공
- 압축 파일 압축/해제/자동 해제 지원
- 테마(다크/라이트/고대비), 언어(한국어/영어) 전환 지원
