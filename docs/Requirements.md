# 복슬Dir (Boksl Dir) - 요구사항

## 프로젝트 개요

**프로젝트명**: 복슬Dir (Boksl Dir)
**개발 언어**: Rust
**주요 라이브러리**: ratatui, crossterm

## 프로젝트 목적

터미널 환경에서 효율적인 파일 관리 경험을 제공하는 듀얼 패널 파일 매니저

## 요구사항 목록

### 1. UI/UX
- TUI (Text User Interface)
- 듀얼 패널 레이아웃
- 터미널 환경에서 최적화된 UX 제공
- 터미널 윈도우 사이즈별 반응형 UI
- 색상 테마 지원

### 2. 플랫폼 지원
- 멀티 OS 지원 (macOS, Linux, Windows)

### 3. 파일 작업
- 파일/폴더 복사
- 파일/폴더 삭제
- 파일/폴더 이동
- 압축
- 압축 해제
- 일괄 이름 변경
- 파일 선택 (단일/다중)

### 4. 파일 탐색 및 검색
- 파일/폴더 검색 (이름, 내용)
- 파일 정렬 (이름, 크기, 날짜, 확장자)
- 숨김 파일 표시/숨김
- 필터링 (패턴 기반)
- 드라이브/마운트 포인트 빠른 접근

### 5. 내비게이션
- 탭 지원 (여러 디렉토리 동시 탐색)
- 디렉토리 히스토리 (앞으로/뒤로 가기)
- 바로가기
- 북마크/즐겨찾기 관리
- 최근 파일/폴더 접근 이력
- 경로 직접 입력으로 디렉토리 이동
- 경로 입력 시 자동완성 제안 목록 제공
- Ghost text 방식 자동완성 (추천 경로 흐린 색 미리보기)
- Tab 키 자동완성

### 6. 외부 프로그램 연동
- 자주 쓰는 프로그램 북마크
- 북마크 선택시 해당 프로그램 실행
- 확장자별 기본 프로그램 지정
- 파일 열기 프로그램 선택 (여러 옵션 중 런타임 선택)
- OS 기본 연결 프로그램으로 파일 열기
- 파일 연결 프로그램 설정 관리
- 터미널 에디터로 파일 편집 (vi, vim, nano, emacs 등)
- 에디터 종료 후 자동 복귀
- 기본 편집기 설정 가능
- 명령어 실행 (내장 터미널)

### 7. 커스터마이징
- 단축키 지원
- 단축키 커스터마이징
